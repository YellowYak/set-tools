Set

A fast-paced card game of visual perception. Find matching patterns before your opponents do.

How to Play

Each card has four features: color (red, green, purple), shape (oval, diamond, squiggle), count (1, 2, 3), and fill (solid, striped, open).

A Set is three cards where, for each feature, the values are either all the same or all different across the three cards. No feature may have two matching and one different.

  • 12 cards are dealt face-up in a grid.
  • Find a Set, select those 3 cards — they're removed and replaced.
  • If no Set exists, 3 extra cards are added.
  • The game ends when the deck is empty and no Sets remain.
Example Set: 1 purple open squiggle · 1 green striped diamond · 1 red solid oval
Count: all 1. Color: all different. Fill: all different. Shape: all different. ✓
